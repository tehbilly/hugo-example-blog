TOML, YAML, JSON - Oh my! One of the nifty Hugo features we should cover: flexible configuration and front matter formats! This entry has front matter in toml, unlike the last one which used yaml, and json is also available if that’s your preference. The toml front matter used on this entry: +++ title = "Another Hugo Post" description = "Nothing special, but one post is boring." date = "2014-04-09" categories = [ "example", "configuration" ] tags = [ "example", "hugo", "toml" ] +++ This flexibility also extends to your site’s global configuration file.
